# TERRAIN

A subset of [ZDoom](https://zdoom.org/wiki/TERRAIN) `TERRAIN` is supported.

## What does work

### splash

- `smallclass`
- `smallclip`
  - **must** be set to **0**
- `baseclass`
- `chunkclass`
- `chunkxvelshift`
- `chunkyvelshift`
- `chunkzvelshift`
- `chunkbasezvel`
- `sound`

### terrain

- `splash`
- `liquid`
- `friction`
- `damagetype`
- `damageamount`
- `damagetimemask`
- `allowprotection`
